Detailed Description

Class to connect all parts of algorithms.

It provides data to Processors and print penalties. It also generates Processors for every user, object, contact (id) that comes from DataProvider.

ProcessorFactiories are needed to generate Processors for every user (user_id) that comes from DataProvider. Every ProcessorFactiory represents one algorithm.
