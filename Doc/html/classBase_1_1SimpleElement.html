void Base::SimpleElement::Notify ( int  update_counter = 0)
virtual

To define order between elements every element have to know when and how many times it was notified. So every time element is notified this function is used. Different classes uses this function in different way. In Element class Notify has no influence on elements order. In every child class it has.
