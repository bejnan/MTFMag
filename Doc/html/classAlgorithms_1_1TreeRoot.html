Detailed Description

This algorithm similar as Move To Front moves chosen element to first position. Difference is in a way it happens. All elements are stored in full binary tree and every notification moves element up to root node replacing other nodes in it's path. For example, lets tree look like : 

    1
  2   3
 4 5 6 7

Linear order of elements is :
1 3 2 7 6 5 4

After notify element with identifier 5 tree will look : 

    5
  1   3
 4 2 6 7

After moving element to front linear order is :
5 3 1 7 6 4 2
