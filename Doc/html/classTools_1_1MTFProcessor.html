Detailed Description

MTFProcessor is Processor based on simple MTF (Move To Front) algorithm. It uses simples penalty counter, that is Tester class.

Returns penalty "earned" by algorithm. Details depends on penalty counter, but main idea is that penalty is as big as far is user on friend list (algorithm ordered list of elements).
