Detailed Description

This class split responsibility of storage Processors with using Processor. User of Processors is DataCollector. Database store, manages and gives access to Processor for DataCollector by id or algorithms name. ProcessorFactory classes responsible for creating Processors are stored in DataCollector. Important! Database class have no outside memory. All it stores is put there during runtime. After runtime all elements are removed from memory. There should be only one instance of class in program.
