Detailed Description

Class provides data read from input file. It implements DataProvider interface. It reads input file given by path in constructor line by line.

Constructor & Destructor Documentation

Base::FileDataProvider::FileDataProvider ( string  file_path)

Constructor opens file pointed by given path. Important note! Read lines are empty after initialization. To read first line method GoToNextLine have to be called.
