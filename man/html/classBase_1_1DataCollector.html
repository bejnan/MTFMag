Detailed Description

Connects algorithms with data in both ways. It delivers data from input to algorithms and puts results to output. Input and output depends on DataProvider and DataOutput implementations. This class in part of 4 class mechanism responsible for running program :

  • DataCollector - controls data flow
  • JudgeCollector - tests algorithms
  • ResultCollector - manage all mechanisms
  • Database - collects algorithms and manage them
